# Markdown

---

## 1. Headings

**Syntax:**

```markdown
# Heading 1

## Heading 2

### Heading 3
```

**Preview:**

# Heading 1

## Heading 2

### Heading 3

---

## 2. Emphasis

**Syntax:**

```markdown
_italic_ or _italic_
**bold** or **bold**
**_bold italic_** or **_bold italic_**
```

**Preview:**  
_italic_ or _italic_  
**bold** or **bold**  
**_bold italic_** or **_bold italic_**

---

## 3. Lists

**Syntax (Unordered):**

```markdown
- Item 1
- Item 2
  - Subitem

* Item 3
```

**Preview:**

- Item 1
- Item 2
  - Subitem

* Item 3

**Syntax (Ordered):**

```markdown
1. First item
2. Second item
   1. Subitem
```

**Preview:**

1. First item
2. Second item
   1. Subitem

---

## 4. Links

**Syntax:**

```markdown
[Link Text](https://example.com)
```

**Preview:**  
[Link Text](https://example.com)

---

## 5. Images

**Syntax:**

```markdown
![Alt Text](https://example.com/image.png)
```

**Preview:**  
![Alt Text](https://example.com/image.png)

## 6. Blockquotes

**Syntax:**

```markdown
> This is a blockquote.
>
> > Nested blockquote.
```

**Preview:**

> This is a blockquote.
>
> > Nested blockquote.

## 7. Code

**Syntax (Inline):**

```markdown
Here is some `inline code`.
```

**Preview:**  
Here is some `inline code`.

**Syntax (Block):**
`js
    // Code goes here
    `

**Preview:**

```js
// Code goes here
```

## 9. Tables

**Syntax:**


**Preview:**

| Header 1 | Header 2 |
| -------- | -------- |
| Row 1    | Row 1    |

## 15. HTML in Markdown

**Syntax:**

```markdown
<b>Bold HTML</b> <br />
<img src="https://example.com/image.png" width="100" />
```

**Preview:**  
<b>Bold HTML</b> <br />
<img src="https://example.com/image.png" width="100" />
